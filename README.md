## Инструкция по сборке

✅ Клонировать ветку "main" на локальный компьютер

```
git clone git@github.com:AnastasiiaKunstman/vue-test-task.git

```
✅ Перейти в папку с проектом

```
cd vue-test-task

```
✅ Установить зависимости

```
npm install

```
✅ Запустить проект

```
npm run dev

```
